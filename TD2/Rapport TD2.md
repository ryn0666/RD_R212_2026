# Rapport TD2

## Points clés

### 1. Le CSS révèle la structure

Avant d'écrire le CSS, j'ai lu le HTML tel quel dans le navigateur. Il était déjà assez lisible (titres, sections, listes). Le CSS est venu améliorer, styliser et rendre le tout plus beau (et c'est très satsfaisant je trouve !!)

### 2. Flexbox pour le layout

J'ai utilisé "flex-wrap" sur "#pourquoi" pour que les trois articles passent automatiquement en colonne sur les mobiles par exemple (principe du responsive)

### 3. Cohérence visuelle avec peu de valeurs

J'ai choisi une seule couleur (burlywood) et je l'ai réutilisée dans le header et les bordures de citation, car je trouve que ça donne une cohérence sans avoir à multiplier les décisions de style !

## Utilisation de l'IA

J'ai utilisé Claude pour vérifier si mon usage de "flex-wrap" était correct et si la page était responsive
