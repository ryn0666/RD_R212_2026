# Rapport TD3

## Points clés

### 1. Le modèle événement → état → rendu

Lors de ce TD j'ai compris que chaque interaction suit uns chèma précis ! Par exemple sur Instagram : clic sur ❤️ (événement) -> le like passe à true (état) -> le compteur s'incrémente à l'écran (rendu)

### 2. classList.toggle

Le JS ne gère pas l'apparence directement, il ajoute ou retire une classe et c'est le CSS qui définit ce que cette classe change visuellement. "toggle" est pratique car il fait les deux à la fois : ajoute si absente, retire si présente (c'est très pratique !)
