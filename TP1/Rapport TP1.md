# Rapport TP1

## Points clés

### 1. Flex et Grid

J'ai utilisé Flex pour le header, le hero et la section "à propos", et Grid pour les cartes projets parce que c'est un layout en colonnes qui doit s'adapter tout seul. La différence s'est clarifiée en pratiquant car au départ c'était un peu flou pour moi !

### 2. Les variables CSS

Définir les couleurs et espacements dans "root" m'a évité de me répéter partout. Quand j'ai voulu changer la couleur principale, j'ai modifié une seule ligne ce qui était efficace
