**Rapport de séance — TP1 HTML / CSS**



**1. Points clés**



1\. Utilisation de Flexbox

J’ai appris à utiliser Flexbox pour organiser les éléments dans une page, notamment pour le header et la section hero. Flex permet d’aligner facilement des éléments horizontalement ou verticalement et de gérer l’espace entre eux.



2\. Utilisation de CSS Grid

J’ai utilisé Grid pour créer la grille de projets. Avec `grid-template-columns: repeat(auto-fit, minmax(250px, 1fr))`, la grille devient automatiquement responsive : les cartes s’adaptent à la taille de l’écran sans avoir besoin de plusieurs media queries.



\*\*3. Variables CSS\*\*

Les variables CSS permettent de centraliser les couleurs, les espacements et la typographie. Cela rend le code plus facile à modifier et plus cohérent, car on peut changer une valeur à un seul endroit pour tout le site.



**2. Utilisation de l’IA**



J’ai utilisé ChatGPT pour m’aider à vérifier que mon CSS respectait bien les consignes du TP et pour améliorer légèrement le style du site (couleurs beige et bouton plus arrondi). L’IA m’a aussi aidé à comprendre certaines propriétés CSS comme Flexbox et Grid.



Cependant, j’ai dû adapter certaines réponses car certaines propositions étaient plus complexes que nécessaire pour ce TP. J’ai donc simplifié le code pour qu’il reste lisible et conforme aux consignes de l’exercice.
