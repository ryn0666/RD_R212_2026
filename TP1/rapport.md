# Rapport — TP1

## Points clés

### 1. Flexbox pour les sections principales
J’ai appris à utiliser Flexbox pour organiser les éléments horizontalement et verticalement, comme le header et la section hero. Cela permet d’aligner facilement le logo, la navigation et de centrer le contenu.

### 2. CSS Grid pour la grille de projets
Grid m’a permis de créer une grille responsive pour les cartes projets avec `repeat(auto-fit, minmax(250px, 1fr))`. Les cartes s’adaptent automatiquement à la taille de l’écran sans besoin de multiples media queries.

### 3. Variables CSS
Les variables permettent de centraliser les couleurs, espacements et typographie. Modifier une couleur ou un espacement se fait à un seul endroit et s’applique partout, ce qui rend le code plus propre et maintenable.

## Utilisation de l'IA

### Démarche
J’ai utilisé ChatGPT (modèle GPT-4 via interface web) pour m’aider à vérifier et améliorer mon CSS.  
Objectifs : vérifier que le code respectait les consignes du TP, proposer un style beige simple et arrondir les boutons.

### Apports et limites
L’IA m’a aidé à structurer mon CSS et à comprendre certaines propriétés Flexbox et Grid.  
Certaines propositions étaient plus complexes que nécessaire, j’ai donc simplifié le code pour qu’il reste lisible et conforme aux consignes du TP.

### Prompts significatifs
1. "Peux-tu vérifier si mon CSS respecte les consignes du TP (Flex, Grid, responsive, variables CSS) ?" → a permis de confirmer que tout était correct et de corriger les doublons.
2. "Comment rendre un bouton CSS plus arrondi ?" → utile pour moderniser le style du site.
3. "Comment donner un style beige simple à mon site avec des variables CSS ?" → a permis de choisir une palette cohérente.
4. "Explique-moi simplement comment fonctionne grid-template-columns avec repeat et minmax." → a clarifié le fonctionnement de la grille responsive.
